# 🛠️ Star-Delta Manual Starter – 11kW Motor

This project includes the design and documentation of a manual Star-Delta starter circuit for an 11kW three-phase motor.

## ⚡ Motor Specifications
- Power: 11 kW  
- Voltage: 400V AC  
- Frequency: 50 Hz  
- Power Factor: 0.85  
- Efficiency: 90%
